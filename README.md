- TensorFlow (https://www.tensorflow.org/)
- Keras (https://keras.io/)

- NLTK (https://www.nltk.org/)
- OpenCV (https://opencv.org/)
- Gensim (https://radimrehurek.com/gensim/)
- Dask (https://dask.org/)
